## Overview

While we recommend using [`create-fivem`](/docs/npm-packages/create-fivem) for the smoothest setup experience, you can also manually install and configure the compiler in your existing project. This guide walks through the step-by-step process of setting up `fivem-compiler` from scratch.

The manual installation gives you more control over the initial setup and is useful when integrating the compiler into an existing development workflow or when you need a custom configuration from the start.

## Choosing a Language

The compiler supports three languages:

- JavaScript
- TypeScript
- Lua

Please choose the language you want to use for your project.
