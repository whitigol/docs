## wLauncher

**wLauncher** is a sleek and intuitive FiveM launcher built for simplicity and performance.

### Features

- **User-Friendly**: Designed to be effortless for anyone to set up and use.
- **Fully Customizable**: Tailor the launcher's look and behavior to your preferences.
- **Multi-Server Support**: Add and manage multiple servers with ease.
- **Automatic Updates**: Stay up to date â€” wLauncher checks and installs updates automatically.
- **Nickname Changer**: _The first launcher to support per-server nicknames._ Set a unique name for every server you join.
- **Themes**: Choose from a variety of themes to personalize your experience.
- **Profile Exporting**: Share your server profiles with friends while keeping sensitive server info private.
- **Live Server Info**: View player count and server status before connecting.
- **More to Come**: We're actively adding features â€” stay tuned!
